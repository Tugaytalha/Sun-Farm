# Sun-Farm

Production forcasting for Solar Panels & C2C sale of electricity and Panels. A project aiming to popularize renewable energy sources developed for the sustainability goal “Accessible and Clean Energy” within the BTK - Huawei Coding Marathon. W/@umit-sen and Oguzhan Kaya 

## 📋 Proje Hakkında

Bu proje Production forcasting for Solar Panels & C2C sale of electricity and Panels. A project aiming to popularize renewable energy sources developed for the sustainability goal “Accessible and Clean Energy” within the BTK - Huawei Coding Marathon. W/@umit-sen and Oguzhan Kaya  amacıyla geliştirilmiştir.
